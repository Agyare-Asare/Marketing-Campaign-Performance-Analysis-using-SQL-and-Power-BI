# Marketing-Campaign-Performance-Analysis-using-SQL-and-Power-BI
This project analyzes customer response data from a marketing campaign. The goal focuses on understanding which customer segments respond, spend more, and engage across channels.
Tools Used
## SQL Server for cleaning, transformation, and analysis.
Power BI for data modeling, measures, and dashboards.

## Dataset
Customer demographics, household structure, spending data, engagement metrics, and a campaign response indicator. All personal identifiers were removed.
https://www.kaggle.com/datasets/rodsaldanha/arketing-campaign

## Data Preparation
Fixed incorrect column names.
Created income groups for segmentation.
Calculated total customer spending.
Combined exploratory queries into one analytical SQL view.
Exported one clean dataset for Power BI.

## Key Insights
Higher income customers respond more often.
Responders spend more than non responders.
Web and store engagement links strongly to response.
Education level influences campaign performance.
Spending patterns show a clear signal.
Customers who respond spend more on average.
Non responders spend less across product categories.

Engagement matters.
Responders visit websites more often.
Responders make more web and store purchases.


## Business Value
This analysis supports better targeting, improved campaign efficiency, and stronger focus on high value customers.
